# Консольная игра «Змейка» на Node.js

Классическая игра «Змейка», реализованная на чистом JavaScript с использованием платформы Node.js.

## Описание

Игрок управляет змейкой, которая движется по замкнутому полю 20×20 ячеек. Цель — собирать еду (символ `*`), увеличивая длину змейки и набирая очки. Игра завершается при столкновении со стенами или собственным телом. Реализованы пауза, рестарт, подсчёт очков и постепенное увеличение скорости.

## Функционал

- Движение змейки в четырёх направлениях (стрелки ↑ ↓ ← →)
- Сбор еды с увеличением счёта и длины змейки
- Постепенное ускорение игры (от 200 мс до 80 мс)
- Защита от разворота на 180 градусов
- Пауза (клавиша P)
- Рестарт в любой момент (клавиша R)
- Выход из игры (клавиша Q)
- Экран приветствия с инструкцией
- Экран Game Over с финальным счётом
- Информационная панель (HUD): счёт, длина, скорость
- Кроссплатформенная очистка экрана

## Требования

- Node.js версии 12.0 или выше
- Терминал с поддержкой ANSI escape-кодов
- Операционная система: Windows, macOS или Linux

## Установка и запуск

1. Убедитесь, что Node.js установлен:

    node --version

2. Откройте терминал или PowerShell
    Откройте в нём директорию в которую вы сохранили snake.js и напишите :
    node snake.js
